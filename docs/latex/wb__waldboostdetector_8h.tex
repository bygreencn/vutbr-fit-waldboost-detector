\subsection{Detailed Description}
Waldboost detector. 

Global and device functions (all in wb namespace) and a Waldboost\+Detector class which uses the as gpu kernels for object detection using waldboost metaalgorithm and L\+B\+P features. Only the Waldboost\+Detector class should be used on its own.
