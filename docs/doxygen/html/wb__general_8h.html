Macro Definition Documentation

#define WB_MAX_DETECTIONS   512

Max. number of detections per frame.

Allocates memory and cannot overflow, but can be kept lower to save GPU memory.

#define WB_SCALING_FACTOR   1.09050773f

Downsampling scaling factor.

1.09050773f = 2^(1/8) Should be 2^(1/WB_LEVELS_PER_OCTAVE), because the number of levels says how many downsampled images are created before an image of 1/2 width/height is created. Smaller number would generate bigger images, bigger number smaller images and the last image wouldn't be half the width/height.
